# /ui-create

Create a UI component. **Always ask user first** — they prefer doing frontend work themselves.

Offer three options:
1. **Full implementation** — delegate to Bard for complete component with types, styling, tests
2. **Structure only** — create component skeleton with props interface and test file, user adds styling
3. **Guidance only** — recommend architecture, patterns, and approach without writing code

Check project context for styling system (Fela, styled-components, etc.) before implementing.
