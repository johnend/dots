# /implement-with-artificer

Transition from Mentor teaching session to Artificer implementation. Captures full session context for clean handoff.

## Mentor's Workflow

1. **Capture context:** topic discussed, key decisions made, files explored, user's understanding level
2. **Check git status** — note any uncommitted changes
3. **Format handoff message:**

```
@Artificer - Implementation request from Mentor session:

Topic: [what was discussed]
Summary: [key decisions and approach agreed upon]
Files: [relevant files identified during session]
Understanding: [what the user now understands]
Requirements: [what needs to be built]
Constraints: [any limitations discussed]
Git Status: [current state]
```

4. End Mentor session — Artificer takes over with full context
