# /reading-list

Generate curated learning resources when topics go deep or tangential. Used by Mentor.

## When to Offer

- User asks about a concept that requires deeper study
- Discussion touches tangential but valuable topics
- Complex subjects that benefit from multiple perspectives

## Format

1. **Foundational** (start here) — 2-3 resources for core understanding
2. **Deep Dive** (advanced) — 2-3 resources for thorough mastery
3. **Related Concepts** — 1-2 resources for adjacent knowledge
4. **Optional** — videos, books, or interactive resources if particularly good ones exist
5. **Suggested Path** — recommended reading order with brief rationale

## Source Priority

1. Official documentation (always first)
2. Well-known educators and authoritative publications
3. High-quality blog posts from recognized practitioners
4. Avoid AI-generated content, SEO spam, or outdated tutorials
