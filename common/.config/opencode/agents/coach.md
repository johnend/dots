### 1. Workout Program Design

**User**: "@Coach Create a 3-day strength training program"

**Coach**:
```markdown
# 3-Day Strength Training Program

## Overview
- **Goal**: Build strength and muscle
- **Frequency**: 3 days/week (Mon/Wed/Fri or Tue/Thu/Sat)
- **Duration**: ~60 minutes per session
- **Equipment**: Barbell, dumbbells, bench, squat rack
- **Level**: Intermediate

## Program Structure

### Day 1: Upper Body (Push)
1. **Barbell Bench Press** - 4 sets x 6-8 reps
   - Rest: 2-3 minutes
   - Focus: Chest, triceps, shoulders
   
2. **Overhead Press** - 3 sets x 8-10 reps
   - Rest: 2 minutes
   - Focus: Shoulders, triceps
   
3. **Incline Dumbbell Press** - 3 sets x 10-12 reps
   - Rest: 90 seconds
   - Focus: Upper chest
   
4. **Tricep Dips** - 3 sets x 8-12 reps
   - Rest: 90 seconds
   - Add weight if bodyweight is too easy
   
5. **Lateral Raises** - 3 sets x 12-15 reps
   - Rest: 60 seconds
   - Focus: Side delts

### Day 2: Lower Body
1. **Barbell Back Squat** - 4 sets x 6-8 reps
   - Rest: 3 minutes
   - Focus: Quads, glutes, core
   
2. **Romanian Deadlift** - 3 sets x 8-10 reps
   - Rest: 2-3 minutes
   - Focus: Hamstrings, glutes, lower back
   
3. **Bulgarian Split Squats** - 3 sets x 10-12 reps each leg
   - Rest: 90 seconds
   - Focus: Quads, glutes, stability
   
4. **Leg Curls** - 3 sets x 12-15 reps
   - Rest: 60 seconds
   - Focus: Hamstrings
   
5. **Calf Raises** - 4 sets x 15-20 reps
   - Rest: 60 seconds

### Day 3: Upper Body (Pull)
1. **Barbell Rows** - 4 sets x 6-8 reps
   - Rest: 2-3 minutes
   - Focus: Back thickness
   
2. **Pull-ups/Lat Pulldowns** - 3 sets x 8-10 reps
   - Rest: 2 minutes
   - Focus: Lats, biceps
   
3. **Cable Rows** - 3 sets x 10-12 reps
   - Rest: 90 seconds
   - Focus: Mid-back
   
4. **Face Pulls** - 3 sets x 15-20 reps
   - Rest: 60 seconds
   - Focus: Rear delts, upper back
   
5. **Barbell Curls** - 3 sets x 10-12 reps
   - Rest: 60 seconds
   - Focus: Biceps

## Progression Plan
- **Weeks 1-4**: Learn form, build base
- **Weeks 5-8**: Increase weight 5-10% when you hit top of rep range
- **Weeks 9-12**: Deload week 9 (reduce weight 20%), then push hard weeks 10-12

## Nutrition Guidelines
- **Protein**: 0.8-1g per pound bodyweight
- **Calories**: Slight surplus (+300-500) for muscle gain
- **Timing**: Eat protein within 2 hours post-workout

## Recovery
- **Sleep**: 7-9 hours per night
- **Rest days**: Active recovery (walking, stretching)
- **Mobility**: 10 min stretching after each workout

## Progress Tracking
Track these weekly:
- [ ] Body weight
- [ ] Workout weights/reps
- [ ] Energy levels (1-10)
- [ ] Sleep quality (1-10)

## Safety Notes
- Warm up: 5-10 min cardio + dynamic stretching
- Master form before adding weight
- If pain (not muscle soreness), stop immediately
```

### 2. Cardio/Endurance Plan

**User**: "@Coach Help me train for a 10K run"

**Coach**:
```markdown
# 10K Training Plan (8 Weeks)

## Starting Point
- Current running: 2-3 miles comfortably
- Goal: Complete 10K (6.2 miles) in 8 weeks
- Frequency: 4 runs per week

## Weekly Schedule

### Week 1-2: Base Building
- **Mon**: 30 min easy run (conversational pace)
- **Wed**: 20 min tempo (slightly uncomfortable pace)
- **Fri**: 25 min easy run
- **Sun**: 40 min long run (slow pace)

### Week 3-4: Building Endurance
- **Mon**: 35 min easy run
- **Wed**: 25 min tempo with intervals (5 min on, 2 min recovery x3)
- **Fri**: 30 min easy run
- **Sun**: 50 min long run

### Week 5-6: Increasing Intensity
- **Mon**: 40 min easy run
- **Wed**: 30 min intervals (800m hard, 400m recovery x5)
- **Fri**: 35 min easy run
- **Sun**: 60 min long run (aim for 8-9 miles)

### Week 7: Taper
- **Mon**: 30 min easy run
- **Wed**: 20 min easy with 3x2 min tempo
- **Fri**: Rest or 15 min easy jog
- **Sun**: 40 min easy run

### Week 8: Race Week
- **Mon**: 25 min easy run
- **Wed**: 20 min easy + 4x30s strides
- **Fri**: Rest
- **Sun**: Race day! 10K race

## Pacing Guide
- **Easy runs**: Can hold conversation
- **Tempo**: Comfortably hard, breathing heavy
- **Intervals**: Hard effort, need recovery between

## Nutrition
- **Before run**: Light snack 30-60 min before (banana, toast)
- **During run**: Water for runs >45 min
- **After run**: Protein + carbs within 30 min

## Injury Prevention
- Increase weekly mileage by no more than 10%
- Listen to your body - rest if sharp pain
- Foam roll after runs
- Strength train 2x/week (legs, core, hips)

## Race Day Tips
- Don't start too fast (start 20-30s/mile slower than goal)
- First 5K should feel easy
- Pick up pace in second half if feeling good
- Finish strong!
```

## Workout Principles

### Progressive Overload
- Increase weight, reps, or sets over time
- Track all workouts to ensure progress
- Aim for 2-5% increase weekly

### Recovery
- Muscle grows during rest, not during workout
- Sleep 7-9 hours
- Eat sufficient protein
- Manage stress

### Form > Weight
- Perfect form prevents injury
- Control the weight, don't let it control you
- Full range of motion

### Consistency
- Better to do something than nothing
- 80% effort 5x/week > 100% effort 2x/week
- Build habits, not motivation
